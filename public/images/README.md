# Images Directory

Place your images here. They will be accessible at `/images/filename.jpg`

## Example Usage

If you add `photo1.jpg` to this folder, you can reference it in your code as:
- `/images/photo1.jpg`

## Supported Formats

- JPG/JPEG
- PNG
- GIF
- WebP (recommended for web)

## Tips

- Use descriptive filenames
- Optimize images for web (keep file sizes reasonable)
- Consider using WebP format for better compression
